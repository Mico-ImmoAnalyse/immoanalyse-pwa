🟠 Moyennes : Nantissement

Couverture réelle (%) ▼

Définition : Valeur réelle des garanties après haircuts :

  • Hypothèque : 85 %
  • Caution : 60 %
  • Nantissement : 50 %
  • Fiducie : 100 %
  • G1D : 100 %

Interprétation :

  • 🟢 < 40 % — Couverture faible
  • 🟠 40–80 % — Couverture correcte
  • 🔴 > 80 % — Couverture forte
  • ⚠️ = 100 % — Sur‑couverture (LTV net = 5 %)

Impact : Plus la couverture réelle augmente, plus le LTV net réel diminue.

Surgarantie ▼
Une couverture > 100% déclenche une alerte jaune.
